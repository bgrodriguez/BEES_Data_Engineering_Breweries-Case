
# ⚙️ How to Run

-- Through Terminal on Visual Studio Code execute the following steps

1. Clone Repository
   git clone https://github.com/bgrodriguez/BEES_Data_Engineering_Breweries-Case.git
   cd BEES_Data_Engineering_Breweries-Case

2. Start Airflow
   docker compose up

3. Access Airflow UI
   http://localhost:8080
   Credentials:
   admin / admin

4. Trigger Pipeline
   Enable DAG
   Click "Trigger DAG"
